# CHISA: Child Health Integrated Surveillance Assistant
### Digital Public Good (DPG) | UNICEF Venture Fund 2026 Submission

CHISA is an open-source, offline-first mobile architecture engineered to mitigate preventable childhood mortality and morbidity at the intersection of climate change, public health, and socio-economic vulnerabilities.

## 🛠️ Core Technology Stack
- **AI Core:** Gemini-1.5-Flash (Configured for deterministic, zero-hallucination processing)
- **Framework Grounding:** Retrieval-Augmented Generation (RAG) mapped to WHO/iCCM guidelines
- **Backend Integrations:** Google Apps Script / AppSheet Native Architecture
- **Ecosystem Webhooks:** Native bidirectional synchronization pipelines for **DHIS2** and **mWater**

## 📂 Repository Structure
- `/src`: Contains core application logic, system instructions, and structured JSON schema configurations for clinical parsing.
- `/docs`: Project specifications and implementation roadmaps.
